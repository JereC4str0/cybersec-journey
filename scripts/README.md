# Scripts de aprendizaje

Cada carpeta contiene herramientas que voy construyendo para reforzar conceptos.

## Proyectos actuales
- `port-scanner/`: Escáner de puertos TCP en Python.
- `web-fuzzer/`: Fuzzer de directorios/archivos web.

- `post-exploitation/`: Scripts de post-explotación.

## Reglas
- Cada script debe tener su propio README.md.
- Preferir Python 3 o Bash.
- Documentar qué aprendí con cada script.
